# リリースノート

## v1.0.0 — MVP (2026-03-14)

### 追加
- `/startday` — タスク登録・セッション開始
- `/progress` — 進捗報告・Claude判断取得
- `/endday` — セッション終了・サマリー取得
- 無反応検知（60分リマインド / 120分メール / 180分スコープ縮小）
- SMTP メール通知
- Claude Opus 4.6 による構造化JSON判断
- Pydantic による出力スキーマ検証
- ruff による静的解析
- pytest ユニットテスト 39件
- pytest 結合テスト 8件

### 既知の制限
- 状態はインメモリのみ（プロセス再起動でリセット）
- シングルインスタンス構成（水平スケール非対応）
- Claude API 呼び出しが同期処理
- リトライ戦略なし
